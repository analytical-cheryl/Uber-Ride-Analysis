# Uber Ride Analysis

## Project Overview

This project analyzes Uber ride data using Microsoft Excel. The objective was to clean the data, create meaningful visualizations, and generate insights into ride patterns and customer trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## Key Insights
- Ride demand by time of day
- Most common pickup locations
- Revenue trends
- Ride type distribution
- Driver performance
